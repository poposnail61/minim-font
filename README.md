# Minim Font Manager

A web-based tool for managing, splitting, and serving web fonts with optimized CSS generation.

## How to Use

### 1. Hosted CSS URL
You can link directly to the CSS file generated by this manager.

**For HTML:**
```html
<link rel="stylesheet" href="https://your-domain.com/release/[FontID]/[FontName].css" />
```

**For CSS (@import):**
```css
@import url("https://your-domain.com/release/[FontID]/[FontName].css");
```

---

### 2. Supported Fonts
Currently managed fonts and their usage names:

| Font ID (Folder) | Font Family Name | Usage CSS |
| :--- | :--- | :--- |
| `MinimBaseVF` | `MinimBaseVF` | `font-family: "MinimBaseVF";` |
| `MinimSoftVF` | `MinimSoftVF` | `font-family: "MinimSoftVF";` |

> **Note**: The Font ID is usually the lowercase, hyphenated version of the font file name.

### 3. Example Usage

```css
body {
  font-family: "MinimBaseVF", sans-serif;
  font-weight: 400;
}

h1 {
  font-family: "MinimSoftVF", sans-serif;
  font-weight: 700;
}
```

## Features
- **Font Splitting**: Automatically splits large font files using `pyftsubset` (via `fonttools`).
- **Variable Font Support**: Detects and supports variable weight ranges.
- **Interactive Preview**: Test fonts with custom text, size, and weight controls.
- **Release Workflow**: Promote tested fonts to production with a single click (Confirmation Modal included).
- **Auto-deployment**: Automatically pushes changes to GitHub upon release.

## Run Locally

```bash
npm install
npm run dev
```
